# TutorCanino Monorepo

Diretório completo de raças caninas - Plataforma de referência nacional sobre raças de cães no Brasil.

## 🎯 Objetivo

Criar o site de referência nacional sobre raças caninas no Brasil, com informações completas, confiáveis e acessíveis para tutores atuais e futuros de cães.

## 📦 Estrutura do Monorepo

```
tutor-canino-monorepo/
├── apps/
│   └── web/                   # Aplicação web principal (Next.js)
├── packages/
│   ├── ui/                    # Componentes UI compartilhados
│   ├── data/                  # Dados de raças caninas
│   └── content/               # Scripts de geração de conteúdo
├── scripts/                   # Scripts utilitários
└── docs/                      # Documentação
```

## 🚀 Tecnologias

- **Frontend**: Next.js 15, React 19, TypeScript 5, Tailwind CSS 4
- **UI**: Radix UI, Lucide Icons
- **Dados**: 217+ raças caninas
- **Conteúdo**: Google Gemini 2.0 API
- **Deploy**: Vercel
